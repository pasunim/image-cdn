เกมสำหรับ awareness เรื่อง sustainability

goal setting
    1. ออกแบบเกมที่ช่วยให้พนักงาน
        - ตระหนักรู้ (Awareness) เรื่องความยั่งยืน (Sustainability)
        - การใช้ทรัพยากรอย่างคุ้มค่า (พลังงาน, น้ำ, วัสดุ, เวลา)
        - ความยั่งยืนด้านพนักงาน (คน, การทำงานร่วมกัน, Well-being)
    2. ความยั่งยืนขององค์กรในระยะยาว
    3. เข้าใจผลกระทบของพฤติกรรมตนเอง ต่อองค์กรและสิ่งแวดล้อม
    4. เกิดแรงจูงใจในการเปลี่ยนพฤติกรรมจริง ในการทำงานประจำวัน

target behaviors
    1. พนักงานบริษัทเอกชนทุกระดับ
    2. ไม่จำเป็นต้องมีพื้นฐานด้าน Sustainability มาก่อน

select mechanics
    ออกแบบ Mini Game จำนวน 3-4 เกม ภายใต้ธีมเดียวกัน
    1. รูปแบบโดยรวม
        - เป็น เกมตะลุยด่าน (Level-based / Mission-based)
    2. แต่ละด่านมี
        - เนื้อเรื่อง (Narrative)
        - สถานการณ์จำลองในที่ทำงาน
        - คำถาม / การตัดสินใจ / Puzzle
    3. ผู้เล่นสะสมคะแนนจาก
        - การตอบคำถามถูก
        - การตัดสินใจที่ยั่งยืน
        - การผ่านเงื่อนไขพิเศษ
    4. npc ในเกมจะมี
        - ผู้สัมภาษณ์ ตัวละครในการคอยถาม https://github.com/pasunim/image-cdn/blob/main/hrd.png
        - ผู้เล่น ตัวละครหลักในการเดินเรื่อง https://github.com/pasunim/image-cdn/blob/main/office-worker.jpg

feedback & rewards
    1. คะแนน (Points)
        - คะแนนพื้นฐาน ตอบถูก / ผ่านด่าน
    2. Bonus Points
        - เลือกทางที่ส่งผลดีต่อบริษัทระยะยาว
        - ลดทรัพยากร / เพิ่มประสิทธิภาพ
        - มีแนวโน้มเปลี่ยนแปลงพฤติกรรมได้จริง
    3. Badges (ตราสัญลักษณ์)
        ตัวอย่าง
            - Green Starter เริ่มเข้าใจแนวคิด Sustainability
            - Resource Saver ใช้ทรัพยากรอย่างมีประสิทธิภาพ
            - Sustainability Champion คิดเชิงระบบและทีม
            - Impact Creator แนวคิดมีผลต่อองค์กรจริง
    4. Special Rewards
        รางวัลพิเศษเมื่อ
            - ผ่านด่านตามเกณฑ์ “Impact ต่อบริษัท”
            - ได้คะแนนระดับ Threshold ที่กำหนด
        รางวัลอาจเป็น
            - Recognition ภายในองค์กร
            - คะแนนสะสมแลกของรางวัล
            - Leaderboard (รายบุคคล / รายทีม)
